# minha.coop

Landing page do projeto minha.coop, uma iniciativa em formação para construir infraestrutura digital compartilhada.

## Deploy no GitHub Pages

O workflow em `.github/workflows/pages.yml` publica automaticamente o conteúdo deste repositório no GitHub Pages quando há um push na branch `main`.

No GitHub, abra **Settings → Pages** e selecione **GitHub Actions** como origem do deploy. Depois, faça o push para `main` e acompanhe a execução na aba **Actions**.

Antes de publicar, substitua `SEU_LINK_DO_TELEGRAM_AQUI` em `index.html` pelo link real do grupo do Telegram.
